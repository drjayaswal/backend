This is The Linux Branch

ls - listing files in current directory
pwd - showing present working directory
touch <file-name> - creating a file with the provided name
cat <file-name> - showing the content of file
mkdir <folder-name> - creating a folder with the provided name
rm <file-name> - remove/delete the file
rmdir <folder-name> - remove/delete the folder
head <file-name> - show starting line(s) of the file
tail <file-name> - show trailing line(s) of the file
tail -f <file-name> - show current and upcoming trailing line(s) of the file
less <file-name> - showing the content of file via pagination 
cp <file-name> <folder-name> - copy file with the provided name to the provided folder
mv <file-name> <folder-name> - copy/move file with the provided name to the provided folder and also remove the file from the source destination
wc <file-name> - showing count of Line(s) | Word(s) | Byte(s) respectively
<returning-command> | pbcopy - copying the terminal output to clipboard (macOS aka Darwin)
ln -s <fileToBeLink-name> <fileToLink-name> - creates a soft-link of --leftFile with ++rightFile
ln <fileToBeLink-name> <fileToLink-name> - creates a hard-link of --leftFile with ++rightFile
cut < -b for byte> < -c for character> <N - M range> <file-name> - cut the byte/character within the range i.e. N to M from file
<returning-command | tee <file-name> - creating a file with the content taken from the terminal output and shows it on terminal as well
sort <fileToBeSort-name> <newFile-name> - sort every line on basis of first character of the line and stores that sorted data into the provided file
diff <leftFile> <rightFile> - showing the difference with seperate colors between --left file and ++right file
vi <file-name> - editing the file via vim editor in the terminal
grep '<pattern/word>' <file-name> - showing the avaialblity of the pattern/word in the file
df/df -h - showing the details about the disk an human-readable with '-h' flag
ps - showing all the active processes
vailablity, and idu - showing the details about the disk usage
